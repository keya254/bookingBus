### Booking Bus 


## Roles

    - Admin
    - Owner
    - Driver
    - Passenger

## Admin 

    - can block owner 
    - can type cars
    - can add governorate and cities
 
## Owner 

    - can add/delete car
    - can add/delete trip
    - can add/delete driver

## Driver 

   - check booking seats by code

## Passenger 

   - can booking seats in bus
